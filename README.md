# Simple To-Do App

This is a simple To-Do List web application built with HTML, CSS and JavaScript.

## Features

- Add new tasks
- Mark tasks as completed
- Remove tasks from the list
- Dynamic updates using JavaScript DOM manipulation

## Technologies Used

- HTML
- CSS
- JavaScript

## How it works

The user types a task in the input field and clicks the **Add Task** button.  
JavaScript creates a new list element using `createElement()` and adds it to the page with `appendChild()`.

Each task has two buttons:
- **Done** → marks the task as completed
- **Remove** → deletes the task from the list

Event listeners are used to detect when the buttons are clicked.
